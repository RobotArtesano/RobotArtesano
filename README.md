Desarrollador analítico enfocado en orquestar modelos de Machine Learning dentro de arquitecturas web robustas, seguras y escalables. Priorizo la lógica de negocio, la integridad de los datos y el rendimiento del servidor sobre la ornamentación superficial.

---

### ♟️ Filosofía de Desarrollo

En el ajedrez, el **Juego Posicional** consiste en colocar las piezas en las casillas óptimas a largo plazo, preparando una estructura sólida en lugar de buscar ataques rápidos y frágiles. Aplico esta misma doctrina a la ingeniería de software: 
Construyo infraestructuras escalables (Docker, Redis, bases de datos relacionales) y colas de tareas asíncronas para soportar cálculos estadísticos pesados, garantizando que el núcleo del sistema sea inquebrantable.

---

### 🚀 Desarrollo Destacado: Motor Predictivo de Retail

Una plataforma *end-to-end* diseñada para calcular escenarios de pronóstico y reabastecimiento, resolviendo el problema del catálogo no estructurado mediante flujos de datos automatizados.

*   **Apertura (Infraestructura Base):** Arquitectura backend desarrollada en **Flask**, acoplada a **PostgreSQL** mediante SQLAlchemy para garantizar la integridad transaccional (ACID).
*   **Medio Juego (Machine Learning):** Integración de **Prophet** (modelo bayesiano) y algoritmos de bosque aleatorio (Random Forest) para pronosticar demanda. La carga computacional pesada está delegada a *Workers* asíncronos en segundo plano mediante **Redis** y **RQ**, protegiendo el hilo principal del servidor.
*   **Final (Despliegue y Visualización):** Contenerización estricta del entorno mediante **Docker** (Inyección de dependencias y variables en tiempo de ejecución en **Railway**). La interfaz de usuario es responsiva y sobria, utilizando **Bootstrap** e integrando **Plotly** para la interactividad de los gráficos estadísticos.
